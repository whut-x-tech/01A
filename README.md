## 🛠️ 技术栈
- **文档框架**: VitePress 1.6.3
- **可视化组件**: Mermaid/Checklist/CardGrid
- **CI/CD**: GitHub Actions
- **部署**: GitHub Pages

## 🚦 快速开始
```bash
# 安装依赖
pnpm install

# 本地开发
pnpm docs:dev

# 构建生产版本
pnpm docs:build
```
